# Portal HTML5 Acessível

Projeto desenvolvido com foco em boas práticas de HTML5, acessibilidade, semântica e performance, utilizando apenas HTML5 e CSS interno (sem frameworks ou JavaScript).

---

## Sobre o Projeto

Este projeto apresenta uma página web moderna e acessível que demonstra:

- Estrutura semântica completa do HTML5  
- Navegação acessível via teclado  
- Formulário avançado com validações nativas  
- Uso eficiente de mídia (imagem, vídeo e áudio)  
- Layout responsivo com CSS Grid  
- Boas práticas de UX e acessibilidade (a11y)

---

## Funcionalidades

### Estrutura Semântica
Uso correto de elementos como:
- `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`

---

### Acessibilidade
- Skip link (atalho para conteúdo principal)
- Navegação por teclado com foco visível
- Uso de `aria-label` e `aria-describedby`
- Labels corretamente associadas aos inputs
- Contraste adequado de cores

---

### Formulário Avançado
Validações nativas do HTML5:
- `required`  
- `pattern` (telefone)  
- `min`, `max`, `step`  
- Tipos específicos (`email`, `date`, `url`, `number`)  
- `datalist` para sugestões  
- Feedback visual automático (válido/inválido)

---

### Conteúdo Multimídia
- Imagem com `loading="lazy"`  
- Vídeo com controle nativo e `preload="metadata"`  
- Áudio com carregamento sob demanda  
- Uso de `figure` e `figcaption`  

---

### Responsividade
- Layout com CSS Grid  
- Media queries para diferentes tamanhos de tela  
- Adaptação para mobile  

---

### Performance
- Sem uso de bibliotecas externas  
- Carregamento otimizado de mídia  
- CSS leve e organizado  
- Uso de `decoding="async"` em imagens  

---

## Tecnologias Utilizadas

- HTML5  
- CSS3 (interno)
